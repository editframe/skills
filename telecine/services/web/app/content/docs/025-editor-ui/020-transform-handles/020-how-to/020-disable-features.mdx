## Disable Specific Features

**Problem:** You need to disable drag, resize, or rotation for specific use cases.

**Solution:** Use the `enable-drag`, `enable-resize`, and `enable-rotation` attributes.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enableFitScale>
  <div style={{ position: 'relative', width: '100%', height: '400px', background: '#f3f4f6', border: '2px solid #e5e7eb' }}>
    <div id="element-resize-only" style={{ position: 'absolute', left: '100px', top: '100px', width: '200px', height: '150px', background: '#3b82f6', borderRadius: '8px' }}></div>
    <TransformHandles target="#element-resize-only" enableDrag={false} enableResize={true} />
  </div>
</Demonstration>

```html
<!-- Disable dragging, only allow resize -->
<ef-transform-handles 
  target="#element" 
  enable-drag="false"
  enable-resize="true">
</ef-transform-handles>

<!-- Only allow rotation, no drag or resize -->
<ef-transform-handles 
  target="#element" 
  enable-drag="false"
  enable-resize="false"
  enable-rotation="true">
</ef-transform-handles>

<!-- Disable all transforms (useful for read-only mode) -->
<ef-transform-handles 
  target="#element" 
  enable-drag="false"
  enable-resize="false"
  enable-rotation="false">
</ef-transform-handles>
```

**Use cases:**
- Read-only preview mode: disable all features
- Position-only editing: disable resize and rotation
- Size-only editing: disable drag and rotation
