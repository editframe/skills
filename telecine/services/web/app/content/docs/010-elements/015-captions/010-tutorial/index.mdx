## Tutorial: Your First Captions Composition

This tutorial will guide you through the essential concepts of working with captions elements in Editframe. Each step builds on the previous one, teaching you the fundamental skills needed for caption composition.

### Step 1: Display Basic Segment Captions

Let's start with the simplest captions display - showing complete caption segments as they play.

**What you'll learn:** How to display caption segments using the `<ef-captions-segment>` child element.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-blue-900 to-purple-900 relative">
    <Captions captionsScript="basic-captions" className="text-center text-white text-2xl bg-black bg-opacity-60 px-4 py-2 rounded-lg shadow-lg">
      <CaptionsSegment />
    </Captions>
  </Timegroup>
  <script type="application/json" id="basic-captions">{JSON.stringify({
    segments: [
      { start: 0, end: 3, text: "Welcome to our video presentation." },
      { start: 3, end: 6, text: "Today we'll explore caption features." },
      { start: 6, end: 9, text: "Captions enhance accessibility." }
    ],
    word_segments: [
      { start: 0, end: 0.8, text: "Welcome" },
      { start: 0.8, end: 1.1, text: "to" },
      { start: 1.1, end: 1.4, text: "our" },
      { start: 1.4, end: 1.9, text: "video" },
      { start: 1.9, end: 3, text: "presentation." },
      { start: 3, end: 3.5, text: "Today" },
      { start: 3.5, end: 3.8, text: "we'll" },
      { start: 3.8, end: 4.3, text: "explore" },
      { start: 4.3, end: 4.8, text: "caption" },
      { start: 4.8, end: 6, text: "features." },
      { start: 6, end: 6.8, text: "Captions" },
      { start: 6.8, end: 7.5, text: "enhance" },
      { start: 7.5, end: 9, text: "accessibility." }
    ]
  })}</script>
</Demonstration>

**Try it yourself:** Modify the segment text in the script element to see how captions update automatically.

**What you'll learn:** How to use `<ef-captions-active-word>` to highlight words individually.

### Step 3: Create Context Display

Let's show the current word with surrounding context, creating a more complete reading experience.

**What you'll learn:** How to combine `<ef-captions-before-active-word>`, `<ef-captions-active-word>`, and `<ef-captions-after-active-word>` to show context.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>

      <CaptionsBeforeActiveWord className="text-gray-300 text-xl" />
      <CaptionsActiveWord className="text-yellow-300 text-xl" />
      <CaptionsAfterActiveWord className="text-gray-400 text-xl" />

### Step 4: Style with CSS Variables

Now let's use the `--ef-word-seed` CSS variable to create varied, non-repetitive animations for each word.

**What you'll learn:** How to use CSS variables to create unique styling for each word.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup id="animated-captions-demo" mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-indigo-900 to-teal-900 relative">
    <Captions captionsScript="animated-captions" className="text-center">
      <CaptionsActiveWord 
        className="text-3xl font-bold transition-all duration-500 ease-out bg-black bg-opacity-70 px-3 py-2 rounded-lg shadow-lg"
        style={{
          color: "hsl(calc(360 * var(--ef-word-seed)), 80%, 70%)",
          transform: "scale(calc(1 + 0.3 * var(--ef-word-seed))) rotate(calc(var(--ef-word-seed) * 10deg - 5deg))",
          textShadow: "2px 2px 8px rgba(0,0,0,0.8)"
        }}
      />
    </Captions>
  </Timegroup>
  <script type="application/json" id="animated-captions">{JSON.stringify({
    segments: [
      { start: 0, end: 6, text: "Each word gets its own unique animation style." }
    ],
    word_segments: [
      { start: 0, end: 0.5, text: "Each" },
      { start: 0.5, end: 1.0, text: "word" },
      { start: 1.0, end: 1.4, text: "gets" },
      { start: 1.4, end: 1.7, text: "its" },
      { start: 1.7, end: 2.1, text: "own" },
      { start: 2.1, end: 2.6, text: "unique" },
      { start: 2.6, end: 3.3, text: "animation" },
      { start: 3.3, end: 3.8, text: "style." }
    ]
  })}</script>
</Demonstration>

**Try it yourself:** Experiment with different CSS properties using `var(--ef-word-seed)` to create your own animation variations.
