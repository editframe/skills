## Tutorial: Creating Responsive Compositions with Fit Scale

This tutorial will guide you through using ef-fit-scale to make fixed-size compositions responsive. Each step builds on the previous one, teaching you how to adapt compositions to any screen size while maintaining aspect ratio.

### Step 1: Basic Scaling

Let's start with the simplest use case - wrapping a fixed-size element in ef-fit-scale.

**What you'll learn:** How to make a fixed-size element scale to fit its container.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <ResizableContainer initialWidth={400} initialHeight={300}>
    <FitScale style={{ width: '100%', height: '100%' }}>
      <div style={{ width: '1920px', height: '1080px', background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)', display: 'flex', alignItems: 'center', justifyContent: 'center', color: 'white', fontSize: '24px', fontWeight: 'bold' }}>
        1920 × 1080
      </div>
    </FitScale>
  </ResizableContainer>
</Demonstration>

**Try it yourself:** Drag the blue resize handles on the corners and edges of the container. Notice how the inner box scales to fit while maintaining its 16:9 aspect ratio.

### Step 2: Different Aspect Ratios

Now let's see how ef-fit-scale handles different aspect ratios.

**What you'll learn:** How ef-fit-scale scales content with different aspect ratios.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <ResizableContainer initialWidth={300} initialHeight={300}>
    <FitScale style={{ width: '100%', height: '100%' }}>
      <div style={{ width: '1080px', height: '1080px', background: 'linear-gradient(135deg, #f093fb 0%, #f5576c 100%)', display: 'flex', alignItems: 'center', justifyContent: 'center', color: 'white', fontSize: '24px', fontWeight: 'bold' }}>
        1080 × 1080
      </div>
    </FitScale>
  </ResizableContainer>
</Demonstration>

**Try it yourself:** Drag the resize handles to change the container shape. The square content always maintains its 1:1 aspect ratio, scaling to fit entirely within the container.

### Step 3: Understanding Content Detection

ef-fit-scale automatically finds the first content child element. Let's see how it handles different child elements.

**What you'll learn:** How ef-fit-scale identifies which element to scale.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <ResizableContainer initialWidth={400} initialHeight={300}>
    <FitScale style={{ width: '100%', height: '100%' }}>
      <style>{`/* This style tag is ignored */`}</style>
      <div style={{ width: '1920px', height: '1080px', background: 'linear-gradient(135deg, #4facfe 0%, #00f2fe 100%)', display: 'flex', alignItems: 'center', justifyContent: 'center', color: 'white', fontSize: '24px', fontWeight: 'bold' }}>
        Content Element
      </div>
    </FitScale>
  </ResizableContainer>
</Demonstration>

**Try it yourself:** The style tag is ignored, and only the div element is scaled. Drag the resize handles to see how the content scales as the container changes shape.

### Step 4: Real-World Usage

Let's see ef-fit-scale in a practical scenario with nested content.

**What you'll learn:** How ef-fit-scale scales entire element trees as a unit.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <ResizableContainer initialWidth={400} initialHeight={300}>
    <FitScale style={{ width: '100%', height: '100%' }}>
      <div style={{ width: '1920px', height: '1080px', background: '#000', display: 'flex', alignItems: 'center', justifyContent: 'center', position: 'relative' }}>
        <div style={{ width: '80%', height: '60%', background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)', borderRadius: '8px', display: 'flex', alignItems: 'center', justifyContent: 'center', color: 'white', fontSize: '32px', fontWeight: 'bold' }}>
          Composition
        </div>
      </div>
    </FitScale>
  </ResizableContainer>
</Demonstration>

**Try it yourself:** Drag the resize handles to change the container size. Notice how the entire composition scales as a unit, maintaining all internal proportions and relationships.
