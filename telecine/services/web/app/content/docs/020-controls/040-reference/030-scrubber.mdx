## Scrubber Reference

### Quick Reference Table

<ScrubberPropertyReference />

### Properties

<PropertyDocList>
  <PropertyDoc name="target" type="string" domReadable domWritable htmlAttribute="target" defaultValue='""'>
    Optional ID of a temporal element to control. If not provided, uses context from nearest ancestor.

    ```html
    <ef-scrubber target="my-video"></ef-scrubber>
    ```
  </PropertyDoc>
</PropertyDocList>

### CSS Custom Properties

<PropertyDocList>
  <PropertyDoc name="--ef-scrubber-height" type="CSS custom property" defaultValue="4px">
    Height of the scrubber track.
  </PropertyDoc>

  <PropertyDoc name="--ef-scrubber-background" type="CSS custom property" defaultValue="rgb(209 213 219)">
    Background color of the unplayed portion of the track.
  </PropertyDoc>

  <PropertyDoc name="--ef-scrubber-progress-color" type="CSS custom property" defaultValue="rgb(37 99 235)">
    Color of the progress indicator and handle.
  </PropertyDoc>

  <PropertyDoc name="--ef-scrubber-handle-size" type="CSS custom property" defaultValue="12px">
    Size (width and height) of the draggable handle.
  </PropertyDoc>
</PropertyDocList>

### CSS Parts

- `scrubber` - Track container
- `progress` - Progress indicator
- `handle` - Draggable handle
