## Build Control Bar

**Problem:** You need a complete control bar with multiple controls working together.

**Solution:** Combine multiple control elements within `ef-controls` or directly in preview.

```html
<ef-preview id="player">
  <ef-timegroup slot="canvas" mode="contain" className="w-[1920px] h-[1080px]">
    <ef-video src="/video.mp4"></ef-video>
  </ef-timegroup>
</ef-preview>

<ef-controls target="player">
  <div className="flex items-center gap-4 p-4 bg-gray-900 rounded">
    <ef-toggle-play>
      <button slot="play" className="px-4 py-2 bg-blue-600 text-white rounded">▶</button>
      <button slot="pause" className="px-4 py-2 bg-orange-600 text-white rounded">⏸</button>
    </ef-toggle-play>
    <ef-time-display className="text-white"></ef-time-display>
    <ef-scrubber className="flex-1"></ef-scrubber>
    <ef-toggle-loop>
      <button className="px-4 py-2 bg-gray-700 text-white rounded">🔁</button>
    </ef-toggle-loop>
  </div>
</ef-controls>
```

**Layout tips:** Use flexbox for spacing. Set `flex-1` on scrubber to fill available space.

**Related:** See individual control guides: [Play/Pause](/docs/controls/how-to/use-play-pause-button), [Scrubber](/docs/controls/how-to/use-scrubber), [Time Display](/docs/controls/how-to/use-time-display), [Loop](/docs/controls/how-to/use-loop-button).
