Welcome to the Editframe blog. We'll be sharing thoughts on programmatic video, developer tooling, and what we're building here.

## What is Editframe?

Editframe is a video rendering API for developers. Instead of editing video by hand, you compose it in code — using HTML, React, or our JavaScript SDK — and we render it.
