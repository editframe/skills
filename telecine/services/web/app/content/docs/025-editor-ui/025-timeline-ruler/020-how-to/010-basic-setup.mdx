## Basic Setup

**Problem:** You need to display time markers on a timeline for a specific duration.

**Solution:** Use `ef-timeline-ruler` with `duration-ms`, `zoom-scale`, and `container-width` attributes.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div style={{ position: 'relative', width: '1000px', height: '32px', border: '1px solid #e5e7eb' }}>
    <ef-timeline-ruler
      duration-ms="10000"
      zoom-scale="1.0"
      container-width="1000"
    ></ef-timeline-ruler>
  </div>
</Demonstration>

**Key properties:**
- `duration-ms`: Total timeline duration in milliseconds (e.g., 10000 for 10 seconds)
- `zoom-scale`: Zoom multiplier (1.0 = normal, 2.0 = 2x zoom, 0.5 = zoomed out)
- `container-width`: Content width in pixels (should match your timeline content width)

The ruler automatically calculates optimal marker intervals based on these values. See [Marker Interval Calculation](/docs/editor-ui/timeline-ruler/explanation/marker-interval-calculation) for details.
