# Packages

Editframe publishes a set of packages that you can use to create videos or integrate with the Editframe API.


- `@editframe/api` - The Editframe API client library.
- `@editframe/elements` - custom html elements library, timelines, audio, video, etc. are implement as custom elements.
- `@editframe/react` - 🎖️ React components, hooks, and other utilities to make building videos with Editframe and React a first-class experience.
- `@editfram/assets` - Functions for processing/preparing media.
- `@editframe/cli` - Editframe CLI tool for creating and managing projects, and interacting with the Editframe API.
