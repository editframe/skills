## Handle Zoomed Canvas

**Problem:** Your editor has zoom functionality, and transform handles need to work correctly at different zoom levels.

**Solution:** Set `canvas-scale` to match your zoom level.

```html
<!-- 50% zoom -->
<ef-transform-handles 
  target="#element" 
  canvas-scale="0.5">
</ef-transform-handles>

<!-- 200% zoom -->
<ef-transform-handles 
  target="#element" 
  canvas-scale="2">
</ef-transform-handles>

<!-- Dynamic zoom -->
<script>
  const handles = document.querySelector('ef-transform-handles');
  let zoomLevel = 1;
  
  function updateZoom(newZoom) {
    zoomLevel = newZoom;
    handles.canvasScale = newZoom;
  }
  
  // Update when zoom changes
  updateZoom(1.5);
</script>
```

The `canvas-scale` value converts screen coordinates to canvas coordinates. Set it to match your CSS transform scale or zoom factor. Default is 1 (no scaling).
