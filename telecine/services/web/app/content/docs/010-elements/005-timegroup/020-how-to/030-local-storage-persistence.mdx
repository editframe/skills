## Use LocalStorage Persistence

### Problem

You want users to resume playback from where they left off after reloading the page.

### Solution

Add an `id` attribute to your root timegroup. The system automatically saves and restores `currentTime`.

```html
<ef-timegroup id="my-video" mode="fixed" duration="60s">
  <!-- Will remember playback position across page reloads -->
</ef-timegroup>
```

**Storage key**: `ef-timegroup-${id}`

**Automatic behavior**:
- Saves `currentTime` after each seek
- Restores `currentTime` on page load (if key exists)
- Works only for root timegroups with `id` attribute
