# @editframe/create

`@editframe/create` is a command-line tool for generating new projects based on predefined templates. It provides an interactive interface to select a template and create a new project directory.

## Installation

You can install `@editframe/create` globally using npm:

```bash
npm install -g @editframe/create@beta
```

Or use it directly with npx:

```bash
npx @editframe/create@beta
```

## Usage

To use `@editframe/create`, simply run the command in your terminal:

```bash
create-editframe
```

Or if using npx:

```bash
npx @editframe/create
```

The tool will guide you through the following steps:

1. Enter the name of the directory to generate your project into.
2. Choose a starter template from the available options.

## Features

- Interactive command-line interface
- Multiple starter templates to choose from
- Automatic copying of template files to the target directory

## How It Works

1. The tool first lists all available starter templates.
2. It then prompts you to enter a name for your project directory and select a template.
3. If the target directory already exists, you'll be asked if you want to overwrite it.
4. The selected template is copied to the target directory.
5. After successful creation, the tool provides instructions for next steps.

## Next Steps After Project Creation

After creating your project, follow these steps:

1. Navigate to your project directory:
   ```bash
   cd your-project-name
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start your project:
   ```bash
   npm start
   ```
