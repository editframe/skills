## Choose Between Intrinsic and Trimmed Duration Modes

**Problem:** You have a trimmed video and need to decide whether thumbnails should show the trimmed portion or the entire source video.

**Solution:** Use `useIntrinsicDuration` to control which timeline the thumbnail strip uses.

**Use trimmed mode (default) when:**
- You want thumbnails that match what the user sees in playback
- The trimmed portion is the relevant content
- You're building a scrubber for the trimmed timeline

**Use intrinsic mode (`useIntrinsicDuration="true"`) when:**
- You want to show thumbnails from the entire source video
- Users need to see content outside the trimmed portion
- You're building a preview interface for the full source

**Note:** Custom `startTimeMs` and `endTimeMs` values override `useIntrinsicDuration` and define their own time range.

For technical details on timeline coordinate handling, see the [Timeline Coordinates explanation](/docs/elements/thumbnail-strip/explanation/timeline-coordinates).
