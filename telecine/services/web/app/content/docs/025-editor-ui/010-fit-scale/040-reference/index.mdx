## Reference

Complete technical reference for ef-fit-scale component behavior.

**New to ef-fit-scale?** Start with the [Tutorial](/docs/editor-ui/fit-scale/tutorial) to learn the basics, or explore [How-To Guides](/docs/editor-ui/fit-scale/how-to/) for specific tasks.

### Overview

ef-fit-scale is an automatic scaling container that preserves aspect ratio. It has no properties or attributes—it's a container-only component that scales its content child to fit the container while maintaining aspect ratio.

### Behavior

**Content Detection:**
- Finds the first content child element
- Excludes non-content elements: `style`, `script`, `meta`, `link`, `title`, `noscript`
- Excludes elements with `display: none` or `display: contents`
- Handles slot elements by traversing to assigned content

**Scaling:**
- Calculates scale factor based on container and content aspect ratios
- Uses contain behavior: scales to fit content entirely within container
- Chooses scale factor that ensures content fits (smaller of width/height ratios)
- Centers scaled content using CSS transforms

**Updates:**
- Updates continuously via requestAnimationFrame
- Responds to container size changes
- Cleans up transforms when disconnected from DOM

### Usage

```html
<ef-fit-scale style="width: 100%; height: 400px;">
  <ef-timegroup mode="contain" class="w-[1920px] h-[1080px]">
    <ef-video src="video.mp4" class="w-full h-full"></ef-video>
  </ef-timegroup>
</ef-fit-scale>
```

```jsx
<FitScale style={{ width: '100%', height: '400px' }}>
  <Timegroup mode="contain" className="w-[1920px] h-[1080px]">
    <Video src="video.mp4" className="w-full h-full" />
  </Timegroup>
</FitScale>
```
