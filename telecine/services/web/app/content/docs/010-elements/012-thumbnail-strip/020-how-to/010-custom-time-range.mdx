## Show Thumbnails for Specific Time Ranges

**Problem:** You want to display thumbnails from a specific portion of the video, not the entire duration.

**Solution:** Use `startTimeMs` and `endTimeMs` to define a custom time window for thumbnail generation.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <div className="flex flex-col gap-4">
    <Video 
      id="howto-time-range-video"
      src="https://assets.editframe.com/bars-n-tone.mp4" 
      className="aspect-[16/9]" 
    />
    <ThumbnailStrip 
      target="howto-time-range-video" 
      thumbnailWidth={80}
      startTimeMs={2000}
      endTimeMs={8000}
      className="h-16 rounded border border-gray-600"
    />
  </div>
</Demonstration>

**Steps:**
1. Set `startTimeMs` to the beginning of your desired time window (in milliseconds)
2. Set `endTimeMs` to the end of your desired time window (in milliseconds)
3. The thumbnail strip will generate thumbnails only for this time range

**Timeline behavior:**
- In trimmed mode (default): `startTimeMs` and `endTimeMs` are relative to the trimmed timeline (0ms = start of trimmed portion)
- In intrinsic mode (`useIntrinsicDuration="true"`): `startTimeMs` and `endTimeMs` are relative to the source timeline (0ms = start of source media)

For details on timeline coordinate handling, see the [Timeline Coordinates explanation](/docs/elements/thumbnail-strip/explanation/timeline-coordinates).
