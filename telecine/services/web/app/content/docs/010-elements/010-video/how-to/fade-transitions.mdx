## How to Add Fade Transitions

**Problem:** You need smooth entry and exit fades for your video.

**Solution:** Use CSS animations with the `--ef-duration` CSS variable to time your fade-out relative to the video's end.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup mode="contain" className="w-[720px] h-[480px] flex items-center justify-center bg-black overflow-hidden">
    <Video
      src="https://assets.editframe.com/bars-n-tone.mp4"
      className="size-full object-contain"
      style={{
        animation: "2s fade-in, 2s fade-out calc(var(--ef-duration) - 2s)"
      }}
    />
  </Timegroup>
  <style>{`@keyframes fade-in {
    0% {
      opacity: 0;
    }
    100% {
      opacity: 1;
    }
  }

  @keyframes fade-out {
    0% {
      opacity: 1;
    }
    100% {
      opacity: 0;
    }
  }`}</style>
</Demonstration>

**Reusable pattern:** The `--ef-duration` variable is automatically set on all elements and represents the element's duration in seconds with the `s` suffix (e.g., `10s`).
