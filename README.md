# Editframe Skills

Agent Skills for video composition with Editframe Elements.

## Installation

### Cursor

Add to your project's `.cursor/skills/` directory:

```bash
cp -r skills/elements-composition .cursor/skills/
```

### Claude Code

```bash
/plugin marketplace add editframe/skills
```

### Manual

Copy the `skills/elements-composition/` folder to your agent's skills directory.

## Available Skills

- **elements-composition** - Create video compositions with Editframe Elements
