## Connect Captions to Media Elements

Synchronize captions with video or audio playback using the `target` attribute.

### Solution

Set `target` to the ID of your video or audio element:

```html
<ef-video id="my-video" src="video.mp4" />
<ef-captions target="my-video">
  <ef-captions-segment />
</ef-captions>
```

Captions synchronize with target playback and load transcription data automatically.

**Standalone captions:** Omit `target` to run captions independently within a timegroup:

```html
<ef-timegroup mode="contain">
  <ef-captions captions-script="my-captions">
    <ef-captions-segment />
  </ef-captions>
</ef-timegroup>
```
