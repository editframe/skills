## Demonstration

### Word-by-Word Animation

Animate text word by word with declarative content:

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <style>{`
    @keyframes bounceIn {
      0% {
        transform: scale(calc(0.6 + var(--ef-seed, 0.5) * 0.3)) rotate(calc(-10deg + var(--ef-seed, 0.5) * 20deg));
      }
      50% {
        transform: scale(calc(1.05 + var(--ef-seed, 0.5) * 0.2)) rotate(calc(-3deg + var(--ef-seed, 0.5) * 6deg));
      }
      100% {
        transform: scale(1) rotate(0deg);
      }
    }
    .bounce-in {
      animation: 0.3s ease-out 0s 1 normal none running bounceIn;
    }
  `}</style>
  <Timegroup mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-blue-900 to-purple-900 relative">
    <Text split="word" stagger="50ms" easing="ease-out" duration="4s" className="text-center text-white text-2xl bg-black bg-opacity-60 px-4 py-2 rounded-lg shadow-lg">
      <template>
        <TextSegment className="bounce-in" />
      </template>
      Hello world this is a word-by-word animation test
    </Text>
  </Timegroup>
</Demonstration>

### Character-by-Character Typewriter

Create a typewriter effect with character splitting:

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <style>{`
    @keyframes bounceIn {
      0% {
        transform: scale(calc(0.6 + var(--ef-seed, 0.5) * 0.3)) rotate(calc(-10deg + var(--ef-seed, 0.5) * 20deg));
      }
      50% {
        transform: scale(calc(1.05 + var(--ef-seed, 0.5) * 0.2)) rotate(calc(-3deg + var(--ef-seed, 0.5) * 6deg));
      }
      100% {
        transform: scale(1) rotate(0deg);
      }
    }
    @keyframes typewriter {
      from { opacity: 0; }
      to { opacity: 1; }
    }
    .typewriter {
      opacity: 0;
      animation: typewriter 0.1s ease-out, bounceIn 0.3s ease-out 0s 1 normal none running;
    }
  `}</style>
  <Timegroup mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-green-900 to-blue-900 relative">
    <Text split="char" stagger="50ms" duration="6s" className="text-center text-white text-3xl font-mono">
      <template>
        <TextSegment className="text-yellow-300 typewriter" />
      </template>
      TYPEWRITER
    </Text>
  </Timegroup>
</Demonstration>

### Seed-Based Variation

Use the `--ef-seed` CSS variable to create pseudo-random effects per segment that are deterministic at render time. Each segment gets a unique seed value (0-1) based on its index, enabling varied colors, scales, and rotations while ensuring consistent rendering:

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <style>{`
    @keyframes fadeIn {
      from {
        opacity: 0;
        transform: translateY(20px);
      }
      to {
        opacity: 1;
        transform: translateY(0);
      }
    }
    .seed-variation {
      animation: fadeIn 0.5s ease-out;
    }
  `}</style>
  <Timegroup mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-purple-900 to-pink-900 relative">
    <Text split="char" stagger="50ms" easing="ease-out" duration="3s" className="text-center">
      <template>
        <TextSegment 
          className="text-4xl font-bold seed-variation"
          style={{
            color: "hsl(calc(360 * var(--ef-seed)), 80%, 70%)",
            transform: "scale(calc(1 + 0.2 * var(--ef-seed))) rotate(calc(var(--ef-seed) * 20deg - 10deg))",
          }}
        />
      </template>
      VARIATION
    </Text>
  </Timegroup>
</Demonstration>

### Doubled Characters (Template Pattern)

Use the template pattern to create multiple segments per character, word, or line. This example shows a shadow effect where each character has both a shadow and main segment:

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <Timegroup mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-pink-900 to-rose-900 relative">
    <Text split="char" stagger="20ms" duration="4s" className="text-center text-white text-4xl font-bold">
      <template>
        <TextSegment className="text-pink-300/30 blur-sm" style={{ position: "absolute", transform: "translate(2px, 2px)" }} />
        <TextSegment className="text-white relative" />
      </template>
      DOUBLED
    </Text>
  </Timegroup>
</Demonstration>
