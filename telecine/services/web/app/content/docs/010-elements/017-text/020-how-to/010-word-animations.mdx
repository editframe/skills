## Create Word-by-Word Animations

**Problem:** You want to animate text so each word appears sequentially with custom animations.

**Solution:** Use `split="word"` with a `stagger` attribute and style segments with CSS animations.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <style>{`
    @keyframes bounceIn {
      0% {
        transform: scale(0.6) rotate(-10deg);
      }
      50% {
        transform: scale(1.05) rotate(3deg);
      }
      100% {
        transform: scale(1) rotate(0deg);
      }
    }
    .bounce-in {
      animation: bounceIn 0.3s ease-out;
    }
  `}</style>
  <Timegroup mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-gradient-to-r from-blue-900 to-purple-900">
    <Text split="word" stagger="50ms" easing="ease-out" duration="4s" className="text-center text-white text-2xl">
      <template>
        <TextSegment className="bounce-in" />
      </template>
      Hello world this is a word-by-word animation
    </Text>
  </Timegroup>
</Demonstration>

**Key points:**
- Set `split="word"` to split text by word boundaries
- Use `stagger` to control delay between words (e.g., `"50ms"`)
- Apply CSS animations to `TextSegment` elements in the template
- The `easing` attribute controls how stagger delays are distributed

For understanding how word splitting works, see the [Text Splitting explanation](/docs/elements/text/explanation/text-splitting).
