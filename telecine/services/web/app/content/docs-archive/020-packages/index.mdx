## Packages

Editframe provides a number of packages to help you integrate with our API.

- [@editframe/create](/docs/packages/create)
- [@editframe/cli](/docs/packages/cli)

## Installation

You can install the packages globally using npm:

```bash
npm install -g @editframe/create@beta
npm install -g @editframe/cli
```

Or use them directly with npx:

```bash
npx @editframe/create@beta
npx @editframe/cli
```
