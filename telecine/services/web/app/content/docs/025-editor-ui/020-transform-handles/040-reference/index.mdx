## Reference

Complete technical reference for all transform handles properties and events.

**New to transform handles?** Start with the [Tutorial](/docs/editor-ui/transform-handles/tutorial) to learn the basics, or explore [How-To Guides](/docs/editor-ui/transform-handles/how-to/) for specific tasks.

### Grouped Attributes

#### Transform

<PropertyDocList>
  <PropertyDoc name="bounds" type="TransformBounds" domReadable domWritable>
    The position, size, and optional rotation of the target element. Object with `x`, `y`, `width`, `height`, and optional `rotation` properties.
    
    ```typescript
    interface TransformBounds {
      x: number;
      y: number;
      width: number;
      height: number;
      rotation?: number;
    }
    ```
    
    Set this property to programmatically position/size elements. Read this property to get current transform state. The component updates this property during drag/resize/rotate operations.
  </PropertyDoc>
</PropertyDocList>

#### Features

<PropertyDocList>
  <PropertyDoc name="enableDrag" type="boolean" domReadable domWritable htmlAttribute="enable-drag" defaultValue="true">
    Enable or disable dragging the element by its border. When `false`, the drag area is not interactive.
    
    Default: `true`
  </PropertyDoc>

  <PropertyDoc name="enableResize" type="boolean" domReadable domWritable htmlAttribute="enable-resize" defaultValue="true">
    Enable or disable resize handles. When `false`, the eight corner and edge handles are hidden and non-interactive.
    
    Default: `true`
  </PropertyDoc>

  <PropertyDoc name="enableRotation" type="boolean" domReadable domWritable htmlAttribute="enable-rotation" defaultValue="false">
    Enable or disable the rotation handle. When `true`, a green rotation handle appears above the element.
    
    Default: `false`
  </PropertyDoc>
</PropertyDocList>

#### Configuration

<PropertyDocList>
  <PropertyDoc name="target" type="string" domReadable domWritable htmlAttribute="target">
    CSS selector string identifying the element to attach transform handles to. The component tracks this element's position and size, and updates it when transforms occur.
    
    Example: `"#my-element"` or `".editable-item"`
  </PropertyDoc>

  <PropertyDoc name="canvasScale" type="number" domReadable domWritable htmlAttribute="canvas-scale" defaultValue="1">
    Scale factor for converting between screen coordinates and canvas coordinates. Set this to match your editor's zoom level or CSS transform scale.
    
    Default: `1` (no scaling)
    
    Example: For 50% zoom, set to `0.5`. For 200% zoom, set to `2`.
  </PropertyDoc>
</PropertyDocList>

#### Constraints

<PropertyDocList>
  <PropertyDoc name="minSize" type="number" domReadable domWritable htmlAttribute="min-size" defaultValue="10">
    Minimum width and height in pixels. When resizing, dimensions are clamped to this minimum value.
    
    Default: `10`
    
    Applies to both width and height independently.
  </PropertyDoc>

  <PropertyDoc name="rotationStep" type="number" domReadable domWritable htmlAttribute="rotation-step">
    Rotation snap increment in degrees. When set, rotation snaps to the nearest multiple of this value.
    
    Example: Set to `15` for 15-degree increments, or `90` for cardinal directions.
    
    When not set, rotation is continuous without snapping.
  </PropertyDoc>
</PropertyDocList>

### Events

#### `bounds-change`

Dispatched when the element's position or size changes (drag or resize operations).

**Event detail:**
```typescript
{
  bounds: TransformBounds;
}
```

**Example:**
```javascript
handles.addEventListener('bounds-change', (e) => {
  const { bounds } = e.detail;
  console.log(`Position: ${bounds.x}, ${bounds.y}`);
  console.log(`Size: ${bounds.width} × ${bounds.height}`);
});
```

#### `rotation-change`

Dispatched when the element's rotation changes. Only dispatched when `enableRotation` is `true`.

**Event detail:**
```typescript
{
  rotation: number;
}
```

**Example:**
```javascript
handles.addEventListener('rotation-change', (e) => {
  const { rotation } = e.detail;
  console.log(`Rotation: ${rotation}°`);
});
```

### CSS Custom Properties

Customize handle appearance using CSS variables:

- `--ef-transform-handles-border-color`: Border color of the overlay (default: `#3b82f6`)
- `--ef-transform-handles-dragging-border-color`: Border color when dragging (default: `#2563eb`)
- `--ef-transform-handles-handle-color`: Background color of resize handles (default: `white`)
- `--ef-transform-handles-handle-border-color`: Border color of resize handles (default: `#3b82f6`)
- `--ef-transform-handles-rotate-handle-color`: Background color of rotation handle (default: `#10b981`)

**Example:**
```css
ef-transform-handles {
  --ef-transform-handles-border-color: #8b5cf6;
  --ef-transform-handles-handle-color: #f3f4f6;
}
```
