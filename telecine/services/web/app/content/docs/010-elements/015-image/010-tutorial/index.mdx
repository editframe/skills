## Tutorial: Your First Image Composition

This tutorial will guide you through the essential concepts of working with image elements in Editframe. Each step builds on the previous one, teaching you the fundamental skills needed for image composition.

### Step 1: Display a Basic Image

Let's start with the simplest possible image composition - displaying a single image in a timegroup.

**What you'll learn:** How to place an image element in a timegroup container.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup mode="fixed" duration="5s" className="w-[960px] h-[540px] bg-black flex overflow-hidden">
    <Image className="w-full h-full object-cover absolute top-0 left-0" src="https://assets.editframe.com/bridge.jpg"/>
  </Timegroup>
</Demonstration>

**Try it yourself:** Change `object-cover` to `object-contain` in the className to see how the image fills the container differently.

### Step 2: Set Fixed Duration on Image

By default, an image expands to fill its parent timegroup's duration. Let's learn how to give an image its own fixed duration.

**What you'll learn:** How to use the `duration` attribute to control image display time.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup mode="contain" className="w-[960px] h-[540px] bg-black overflow-hidden">
    <Image duration="5s" className="w-full h-full object-contain object-center absolute top-0 left-0" src="https://assets.editframe.com/bridge.jpg"/>
  </Timegroup>
</Demonstration>

**Try it yourself:** Change the duration from `"5s"` to `"3s"` to see how the image displays for a shorter time.

### Step 3: Create an Image Sequence

Now let's create a simple slideshow by sequencing multiple images with fixed durations.

**What you'll learn:** How to use timegroup `sequence` mode with multiple images to create a slideshow.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup mode="sequence" className="w-[540px] h-[540px] bg-black overflow-hidden">
    <Image duration="5s" className="w-full h-full object-contain object-center absolute top-0 left-0" src="https://assets.editframe.com/dog.jpg"/>
    <Image duration="5s" className="w-full h-full object-contain object-center absolute top-0 left-0" src="https://assets.editframe.com/meme.jpg"/>
  </Timegroup>
</Demonstration>

**Try it yourself:** Add a third image to the sequence with a different duration to see how the timegroup adapts.

### Step 4: Combine Images with Other Content

In practice, you'll often want to combine images with text, shapes, or other elements. Let's wrap each image in its own timegroup to create richer compositions.

**What you'll learn:** How to create complex slideshows by nesting timegroups and combining images with other content.

<Demonstration layout="horizontal" defaultSourceVisible alwaysShowSource>
  <Timegroup id="rich-slideshow" mode="sequence" className="w-[540px] h-[540px] bg-black overflow-hidden">
    <Timegroup mode="fixed" duration="5s" className="w-full h-full bg-black">
      <Image className="w-full h-full object-contain object-center absolute top-0 left-0" src="https://assets.editframe.com/dog.jpg"/>
      <h1 className="bg-blue-900 text-white text-center absolute p-2 bottom-0 left-0 w-full z-10">Breaking news: Handsome boy!</h1>
    </Timegroup>
    <Timegroup mode="fixed" duration="5s" className="w-full h-full bg-black grid grid-cols-[1fr_1fr] grid-rows-[1fr_1fr]">
      <Image className="w-full h-full object-contain object-center absolute top-0 left-0" src="https://assets.editframe.com/meme.jpg"/>
      <h1 className="text-black col-start-2 row-start-1 w-1/4 self-center pl-8 z-10">Image Sequence</h1>
      <h1 className="text-black col-start-2 row-start-2 w-1/4 self-center pl-8 z-10">Timegroup Sequence w/Images</h1>
    </Timegroup>
  </Timegroup>
</Demonstration>

**Try it yourself:** Modify the text content or add additional elements like shapes or captions to create your own custom slide composition.
