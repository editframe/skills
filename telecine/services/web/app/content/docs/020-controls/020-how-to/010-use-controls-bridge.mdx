## Use Controls Bridge

**Problem:** You need to place controls outside the preview element, in a different part of your layout.

**Solution:** Use `ef-controls` with the `target` attribute to bridge context from a remote preview.

```html
<ef-preview id="my-preview">
  <ef-timegroup slot="canvas" mode="contain" className="w-[1920px] h-[1080px]">
    <ef-video src="/video.mp4"></ef-video>
  </ef-timegroup>
</ef-preview>

<ef-controls target="my-preview">
  <ef-toggle-play>
    <button slot="play">Play</button>
    <button slot="pause">Pause</button>
  </ef-toggle-play>
  <ef-scrubber></ef-scrubber>
</ef-controls>
```

The `target` attribute references the preview's `id`. Controls inside `ef-controls` work as if they were nested directly in the preview.

**Learn more:** See [Context Bridging](/docs/controls/explanation/context-bridging) for how this works.
