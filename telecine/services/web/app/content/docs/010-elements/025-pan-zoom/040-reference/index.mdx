### Grouped Attributes

#### Transform

<PropertyDocList>
  <PropertyDoc name="x" type="number" domReadable domWritable htmlAttribute="x" defaultValue="0">
    Horizontal pan offset in pixels. Positive values move content to the left, negative values move content to the right. Can be set programmatically or updated via user interaction (dragging or wheel scrolling without modifier keys).

    The value is applied as a CSS transform, so it directly controls the visual position of the content wrapper.
  </PropertyDoc>

  <PropertyDoc name="y" type="number" domReadable domWritable htmlAttribute="y" defaultValue="0">
    Vertical pan offset in pixels. Positive values move content up, negative values move content down. Can be set programmatically or updated via user interaction (dragging or wheel scrolling without modifier keys).

    The value is applied as a CSS transform, so it directly controls the visual position of the content wrapper.
  </PropertyDoc>

  <PropertyDoc name="scale" type="number" domReadable domWritable htmlAttribute="scale" defaultValue="1">
    Zoom scale factor. Values are automatically clamped between 0.1 and 5. A scale of 1 represents no zoom, values greater than 1 zoom in, and values less than 1 zoom out.

    When zooming via wheel with modifier keys (Ctrl/Cmd), the scale updates automatically while maintaining the pointer position as the zoom center. See [Zoom-to-Pointer Behavior](/docs/elements/pan-zoom/explanation/zoom-to-pointer) for details.
  </PropertyDoc>
</PropertyDocList>

### Events

#### `transform-changed`

Fired whenever the transform changes, either through user interaction or programmatic property updates.

**Event Detail:**
```typescript
{
  x: number;
  y: number;
  scale: number;
}
```

The event detail contains the current transform values. This event bubbles and is composed, so it can be listened to on parent elements.

```jsx
<PanZoom
  onTransformChanged={(e) => {
    const { x, y, scale } = e.detail;
    // Handle transform change
  }}
>
  <YourContent />
</PanZoom>
```

See [Listen to Transform Changes](/docs/elements/pan-zoom/how-to/listen-transform-changes) for usage examples.
