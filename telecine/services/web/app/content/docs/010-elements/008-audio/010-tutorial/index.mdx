## Tutorial: Your First Audio Composition

This tutorial will guide you through the essential concepts of working with audio elements in Editframe. Each step builds on the previous one, teaching you the fundamental skills needed for audio composition.

### Step 1: Display a Basic Audio Element

Let's start with the simplest possible audio composition - displaying a single audio element.

**What you'll learn:** How to place an audio element in a composition container.

### Step 2: Trim Your Audio

Now let's learn how to show only part of an audio file. Editframe supports two approaches to trimming.

**What you'll learn:** How to use both relative trimming (`trimstart`/`trimend`) and absolute trimming (`sourcein`/`sourceout`).

**Relative trimming** removes time from the start and end. The source audio is 10 seconds long. By setting `trimstart="2s"` and `trimend="2s"`, we remove 2 seconds from each end, resulting in a 6-second clip.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <Timegroup mode="contain" className="aspect-[1/1] w-[500px] h-[500px] bg-black flex items-center justify-center">
    <Audio src="https://assets.editframe.com/card-joker.mp3" trimstart="2s" trimend="2s" />
    <p className="text-white text-2xl">[intentionally left blank]</p>
  </Timegroup>
</Demonstration>

**Absolute trimming** uses exact timestamps. By setting `sourcein="2s"` and `sourceout="4s"`, we show exactly the 2-second window from 2s to 4s in the source.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <Timegroup mode="contain" className="aspect-[1/1] w-[500px] h-[500px] bg-black flex items-center justify-center">
    <Audio src="https://assets.editframe.com/card-joker.mp3" sourcein="2s" sourceout="4s" />
    <p className="text-white text-2xl">[intentionally left blank]</p>
  </Timegroup>
</Demonstration>

**Try it yourself:** Experiment with both approaches. For help choosing which to use, see [How to Choose Between trimStart and sourceIn](/docs/elements/audio/how-to/choose-trimming). For deeper understanding, see the [Trimming Semantics explanation](/docs/elements/audio/explanation/trimming-semantics).

### Step 3: Create a Simple Sequence

Now let's combine multiple audio clips to play one after another.

**What you'll learn:** How to create sequential audio playback using `mode="sequence"` on a timegroup.

**Try it yourself:** Add a fourth audio clip to the sequence. Notice how the timeline automatically extends to accommodate it.
