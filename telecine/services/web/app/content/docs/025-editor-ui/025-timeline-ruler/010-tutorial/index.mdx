## Tutorial: Building a Timeline Ruler

This tutorial guides you through using ef-timeline-ruler to display time markers in timeline editors. Each step builds on the previous one.

### Step 1: Basic Timeline Ruler

**What you'll learn:** How to set up a basic timeline ruler with duration and zoom scale.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div style={{ position: 'relative', width: '1000px', height: '32px', border: '1px solid #e5e7eb' }}>
    <ef-timeline-ruler
      duration-ms="10000"
      zoom-scale="1.0"
      container-width="1000"
    ></ef-timeline-ruler>
  </div>
</Demonstration>

**Try it yourself:** The ruler displays time markers at optimal intervals based on container width and duration.

### Step 2: Scroll Synchronization

**What you'll learn:** How to synchronize the ruler with a scrollable container.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div>
    <div style={{ position: 'relative', width: '500px', height: '32px', borderBottom: '1px solid #e5e7eb' }}>
      <ef-timeline-ruler
        duration-ms="20000"
        zoom-scale="1.0"
        container-width="2000"
        scroll-container-selector="#timeline-content"
      ></ef-timeline-ruler>
    </div>
    <div id="timeline-content" style={{ overflowX: 'auto', width: '500px', height: '200px' }}>
      <div style={{ width: '2000px', height: '200px', background: '#f3f4f6' }}></div>
    </div>
  </div>
</Demonstration>

**Try it yourself:** Scroll the timeline content horizontally. The ruler markers stay aligned with the content.

### Step 3: Frame-Based Markers

**What you'll learn:** How frame markers appear when zoomed in.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div style={{ position: 'relative', width: '1000px', height: '32px', border: '1px solid #e5e7eb' }}>
    <ef-timeline-ruler
      duration-ms="5000"
      zoom-scale="10.0"
      container-width="10000"
      fps="30"
    ></ef-timeline-ruler>
  </div>
</Demonstration>

**Try it yourself:** With zoom-scale set to 10.0, the ruler displays frame markers for frame-accurate editing.

### Step 4: Scroll with Multiple Properties

**What you'll learn:** How to combine scroll synchronization with frame markers.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div>
    <div style={{ position: 'relative', width: '1000px', height: '32px' }}>
      <ef-timeline-ruler
        duration-ms="10000"
        zoom-scale="2.0"
        container-width="2000"
        fps="30"
        scroll-container-selector="#timeline-scroll"
      ></ef-timeline-ruler>
    </div>
    <div id="timeline-scroll" style={{ overflowX: 'auto', width: '1000px', height: '400px' }}>
      <div style={{ width: '2000px', height: '400px' }}></div>
    </div>
  </div>
</Demonstration>

**Try it yourself:** Scroll the timeline content. The ruler markers stay synchronized and show frame markers when zoomed in.

### Step 5: Complete Timeline Editor

**What you'll learn:** How to build a full timeline editor with zoom controls.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div>
    <div style={{ position: 'relative', width: '100%', height: '32px' }}>
      <ef-timeline-ruler
        duration-ms="30000"
        zoom-scale="2.0"
        container-width="6000"
        fps="30"
        scroll-container-selector="#complete-timeline-scroll"
      ></ef-timeline-ruler>
    </div>
    <div id="complete-timeline-scroll" style={{ overflowX: 'auto', width: '100%', height: '400px' }}>
      <div style={{ width: '6000px', height: '400px', background: '#f3f4f6' }}></div>
    </div>
  </div>
</Demonstration>

**Try it yourself:** Scroll the timeline content. Markers automatically switch between time-based and frame-based display based on zoom level.
