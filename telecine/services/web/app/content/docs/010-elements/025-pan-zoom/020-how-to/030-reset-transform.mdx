## Reset Transform to Default

**Problem:** You need to reset the pan-zoom back to its initial position and zoom level.

**Solution:** Set `x`, `y`, and `scale` properties back to their default values (0, 0, 1).

```jsx
const panZoomRef = useRef(null);

function resetTransform() {
  panZoomRef.current.x = 0;
  panZoomRef.current.y = 0;
  panZoomRef.current.scale = 1;
}

<PanZoom ref={panZoomRef} className="w-[720px] h-[480px] border-2 border-gray-300 overflow-hidden">
  <YourContent />
</PanZoom>
<button onClick={resetTransform}>Reset View</button>
```

The transform updates immediately when properties are set, providing smooth reset animation if CSS transitions are applied.
