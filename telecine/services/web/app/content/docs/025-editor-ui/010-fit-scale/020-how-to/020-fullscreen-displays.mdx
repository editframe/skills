## Fullscreen Displays

**Problem:** You need to display a composition in fullscreen mode that scales to fill the screen while maintaining aspect ratio.

**Solution:** Use ef-fit-scale with the Fullscreen API and viewport sizing.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <ResizableContainer initialWidth={400} initialHeight={300}>
    <FitScale style={{ width: '100%', height: '100%' }}>
      <div style={{ width: '1920px', height: '1080px', background: 'linear-gradient(135deg, #4facfe 0%, #00f2fe 100%)', display: 'flex', alignItems: 'center', justifyContent: 'center', color: 'white', fontSize: '24px', fontWeight: 'bold' }}>
        1920 × 1080
      </div>
    </FitScale>
  </ResizableContainer>
</Demonstration>

**HTML structure:**

```html
<div id="fullscreen-container" style="width: 100vw; height: 100vh;">
  <ef-fit-scale style="width: 100%; height: 100%;">
    <ef-timegroup mode="contain" class="w-[1920px] h-[1080px]">
      <ef-video src="video.mp4" class="w-full h-full"></ef-video>
    </ef-timegroup>
  </ef-fit-scale>
</div>
```

**JavaScript for fullscreen:**

```javascript
const container = document.getElementById('fullscreen-container');

// Enter fullscreen
container.requestFullscreen();

// Exit fullscreen
document.exitFullscreen();
```

**CSS for fullscreen:**

```css
#fullscreen-container:fullscreen ef-fit-scale {
  width: 100vw;
  height: 100vh;
}
```

The composition scales to fill the screen while maintaining aspect ratio. See [Aspect Ratio Preservation](/docs/editor-ui/fit-scale/explanation/aspect-ratio-preservation) for scaling behavior details.
