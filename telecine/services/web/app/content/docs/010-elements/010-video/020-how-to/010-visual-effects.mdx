## Apply Visual Effects

**Problem:** You need to apply visual effects to a video using CSS filters.

**Solution:** Use standard CSS filter properties directly on the video element via `className` or `style`.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <Timegroup mode="contain" className="w-[720px] h-[480px] flex items-center justify-center bg-black overflow-hidden">
    <Video src="https://assets.editframe.com/bars-n-tone.mp4" className="blur-sm saturate-200 brightness-150 contrast-150 size-full object-contain" />
  </Timegroup>
</Demonstration>

**Available filters:** `blur`, `brightness`, `contrast`, `saturate`, `grayscale`, `sepia`, `hue-rotate`, `invert`, `opacity`.

```jsx
// Multiple filters
<Video src="video.mp4" className="blur-sm brightness-150 contrast-125" />

// Grayscale
<Video src="video.mp4" className="grayscale" />

// Combine filters
<Video src="video.mp4" className="blur-md saturate-150 brightness-110" />
```

All CSS filters work as expected - the video element renders to a canvas, so standard CSS filter properties apply directly.
