## Use Loop Button

**Problem:** You need to toggle video looping on/off.

**Solution:** Use `ef-toggle-loop` with any clickable element.

```html
<ef-toggle-loop>
  <button>🔁 Loop</button>
</ef-toggle-loop>
```

Clicking the element toggles loop state. The button content is fully customizable.

**Custom styling:**

```html
<ef-toggle-loop>
  <button className="px-4 py-2 bg-gray-700 text-white rounded hover:bg-gray-600">
    🔁 Loop
  </button>
</ef-toggle-loop>
```

**Reference:** See [Toggle Loop Reference](/docs/controls/reference/toggle-loop) for properties.
