## Tutorial: Building Video Controls

This tutorial will guide you through using controls with video elements. Each step builds on the previous one, teaching you how to create flexible control layouts.

### Step 1: Basic Controls Within Preview

Let's start with controls placed directly inside a preview element.

**What you'll learn:** How to place control elements within a preview for direct control.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <Preview id="tutorial-preview-1">
    <Timegroup slot="canvas" mode="contain" className="w-[720px] h-[405px] flex items-center justify-center bg-black overflow-hidden">
      <Video src="https://assets.editframe.com/bars-n-tone.mp4" sourceout="2s" className="size-full object-contain" />
    </Timegroup>
    <div slot="controls" className="flex items-center gap-4 p-4 bg-gray-900">
      <TogglePlay>
        <button slot="play" className="px-4 py-2 bg-blue-600 text-white rounded">▶ Play</button>
        <button slot="pause" className="px-4 py-2 bg-orange-600 text-white rounded">⏸ Pause</button>
      </TogglePlay>
      <Scrubber className="flex-1" />
      <TimeDisplay className="text-white" />
    </div>
  </Preview>
</Demonstration>

**Try it yourself:** Add a `<ToggleLoop>` control to the control bar.
