### Properties

<PropertyDocList>
  <PropertyDoc name="target" type="string" domReadable domWritable htmlAttribute="target" defaultValue='""'>

    <ef-time-display target="my-video"></ef-time-display>

### CSS Parts

- `time` - Time text element

### Format

Displays `M:SS / M:SS` where:
- First time: Current playback position
- Second time: Total duration
- Minutes without leading zeros
- Seconds with leading zeros

**How-to:** See [Use Time Display](/docs/controls/how-to/use-time-display).
