## Create Audio Sequences

**Problem:** You need multiple audio clips to play one after another in sequence.

**Solution:** Place multiple audio elements inside a timegroup with `mode="sequence"`.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enablePreview enableFilmstrip enableFitScale>
  <Timegroup mode="sequence" className="aspect-[1/1] w-[500px] h-[500px] bg-black flex items-center justify-center">
    <Audio src="https://assets.editframe.com/card-joker.mp3" sourcein="0s" sourceout="2s" />
    <Audio src="https://assets.editframe.com/card-joker.mp3" sourcein="8s" />
    <Audio src="https://assets.editframe.com/card-joker.mp3" sourcein="2s" sourceout="4s" />
  </Timegroup>
</Demonstration>

**Key points:**
- Each audio element plays sequentially in the order it appears
- The timeline automatically extends to accommodate all clips
- Use `sourcein`/`sourceout` or `trimstart`/`trimend` to select portions of each clip
- All clips synchronize with the root timegroup's playback control

For understanding how timing works across sequences, see the [Time Coordinates explanation](/docs/elements/audio/explanation/time-coordinates).
