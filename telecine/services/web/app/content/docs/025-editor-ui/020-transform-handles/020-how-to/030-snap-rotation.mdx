## Snap Rotation to Steps

**Problem:** You need rotation to snap to specific angle increments (e.g., 15° or 90°).

**Solution:** Use the `rotation-step` attribute to define the snap increment.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource enableFitScale>
  <div style={{ position: 'relative', width: '100%', height: '400px', background: '#f3f4f6', border: '2px solid #e5e7eb' }}>
    <div id="element-snap" style={{ position: 'absolute', left: '100px', top: '100px', width: '200px', height: '150px', background: '#3b82f6', borderRadius: '8px', transformOrigin: 'center' }}></div>
    <TransformHandles target="#element-snap" enableRotation rotationStep={15} />
  </div>
</Demonstration>

```html
<!-- Snap to 15-degree increments -->
<ef-transform-handles 
  target="#element" 
  enable-rotation
  rotation-step="15">
</ef-transform-handles>

<!-- Snap to 90-degree increments (cardinal directions) -->
<ef-transform-handles 
  target="#element" 
  enable-rotation
  rotation-step="90">
</ef-transform-handles>
```

When `rotation-step` is set, rotation snaps to the nearest multiple of the step value. Try rotating the element above - it snaps to 15-degree increments. Useful for constraining rotation to common angles or creating grid-aligned rotations.
