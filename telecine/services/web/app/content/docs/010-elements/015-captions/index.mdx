### CSS Variables in Action

    <Captions captionsScript="animated-captions" className="text-center">
      <CaptionsActiveWord 
        className="text-3xl font-bold transition-all duration-500 ease-out bg-black bg-opacity-70 px-3 py-2 rounded-lg shadow-lg"
        style={{
          color: "hsl(calc(360 * var(--ef-word-seed)), 80%, 70%)",
          transform: "scale(calc(1 + 0.3 * var(--ef-word-seed))) rotate(calc(var(--ef-word-seed) * 10deg - 5deg))",
          textShadow: "2px 2px 8px rgba(0,0,0,0.8)"
