## Scroll Synchronization

**Problem:** Your timeline ruler needs to stay aligned with scrollable timeline content.

**Solution:** Connect the ruler to the scroll container using `scroll-container-selector` or `scrollContainerRef`.

<Demonstration layout="horizontal" sourceVisible alwaysShowSource>
  <div>
    <div style={{ position: 'relative', width: '500px', height: '32px', borderBottom: '1px solid #e5e7eb' }}>
      <ef-timeline-ruler
        duration-ms="20000"
        zoom-scale="1.0"
        container-width="2000"
        scroll-container-selector="#scroll-demo"
      ></ef-timeline-ruler>
    </div>
    <div id="scroll-demo" style={{ overflowX: 'auto', width: '500px', height: '200px' }}>
      <div style={{ width: '2000px', height: '200px', background: '#f3f4f6' }}></div>
    </div>
  </div>
</Demonstration>

The ruler automatically syncs marker positions with scroll position. Use the ref approach in React for better performance and type safety.
