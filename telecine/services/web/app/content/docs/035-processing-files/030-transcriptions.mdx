# Captions

### API

Once a video file has been uploaded, and processed, a transcription can be generated for it.

You will need to determine the `trackId` of the track you want to transcribe. This should be an audio track. A typical video file might have one audio track, and one video track. But it could have many audio tracks, and you can choose which one you want to transcribe.

You _may_ supply a `trackId` to the transcription function, but it is optional. If you do not supply one, the first audio track found will be used.

If no audio tracks are found, or the supplied `trackId` does not match any of the audio tracks found, the transcription request will fail with a `400` status code.

      <CodeStep>
        <CodeStepLabel title="Create a client">
          Import required functions and set up your API client.
        </CodeStepLabel>
        <CodeStepCode>
        ```ts
        import { Client, transcribeISOBMFFFile } from "@editframe/api/node";
        const EF_TOKEN = "/* Load your API TOKEN */";
        const client = new Client(EF_TOKEN);
        ```
        </CodeStepCode>
      </CodeStep>

      <CodeStep>
        <CodeStepLabel title="Initiate the transcription">
        If the selected track has already had a transcription initiated, this will return the existing record.
        </CodeStepLabel>
        <CodeStepCode>
        ```ts
        const fileId = "/* fileId returned from upload/processing operations */";
        const transcriptionRecord = await transcribeISOBMFFFile(client, fileId);
        ```
        </CodeStepCode>
      </CodeStep>

      <CodeStep>
        <CodeStepLabel title="Wait for the transcription to complete">
          The transcription will be queued, and you can receive progress updates.
        </CodeStepLabel>
        <CodeStepCode>
        ```ts
        const progress = getTranscriptionProgress(client, transcriptionRecord.id);

        for await (const event of progress) {
          console.log(event.data.progress);
          // {
          //   type: "progress";
          //   progress: number; // Between 0 and 1
          // }
        }

        // Alternatively, you may await a promise that resolves when the transcription is complete:
        await progress.whenComplete();
        ```
        </CodeStepCode>
      </CodeStep>
    </CodeSteps>

  <PersistentTab label="PHP (Laravel)">
    <CodeSteps>
      <CodeStep>
        <CodeStepLabel title="Set up dependencies">
        Import required classes and define configuration variables.
        </CodeStepLabel>
        <CodeStepCode>
        ```php
        use Illuminate\Support\Facades\Http;

        $apiToken = 'YOUR_API_TOKEN';
        $baseUrl = 'https://editframe.dev';
        $fileId = 'YOUR_FILE_ID'; // ID of your already processed file
        ```
        </CodeStepCode>
      </CodeStep>

      <CodeStep>
        <CodeStepLabel title="Create transcription">
        Start the transcription process for the file.
        </CodeStepLabel>
        <CodeStepCode>
        ```php
        $response = Http::withToken($apiToken)
            ->post("{$baseUrl}/api/v1/transcriptions", [
                'file_id' => $fileId,
                'track_id' => 0 // Specify the audio track to transcribe
            ]);

        if (!$response->successful()) {
            throw new Exception('Failed to start transcription: ' . $response->body());
        }

        $transcription = $response->json();
        ```
        </CodeStepCode>
      </CodeStep>

      <CodeStep>
        <CodeStepLabel title="Monitor transcription progress">
        Poll the transcription status until complete.
        </CodeStepLabel>
        <CodeStepCode>
        ```php
        $complete = false;
        while (!$complete) {
            $response = Http::withToken($apiToken)
                ->get("{$baseUrl}/api/v1/transcriptions/{$transcription['id']}");
            
            if (!$response->successful()) {
                throw new Exception('Failed to get status: ' . $response->body());
            }

            $status = $response->json();
            
            if ($status['status'] === 'failed') {
                throw new Exception('Transcription failed');
            }
            
            if ($status['status'] === 'complete') {
                $complete = true;
                echo "Transcription complete!\n";
                break;
            }
            
            sleep(2); // Wait 2 seconds before checking again
            echo "Transcribing...\n";
        }
        ```
        </CodeStepCode>
      </CodeStep>
    </CodeSteps>

When using the `<ef-captions>`/`<Captions>` component, you will target the an `<ef-video>`/`<Video>` or `<ef-audio>`/`<Audio>` file. Those elements will automatically connect to the Editframe api to fetch the appropriate transcription data.
