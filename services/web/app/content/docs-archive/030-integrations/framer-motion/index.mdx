#  Framer Motion

## Introduction

Editframe React and Framer Motion are powerful tools for creating dynamic and interactive animations in React applications. Editframe React provides components and hooks for managing timing and animation control, while Framer Motion offers a flexible animation library for React. This guide will walk you through the basics of both libraries and show you how to integrate them for stunning visual effects.

## Getting Started

### Installation

First, you'll need to create a new project using `@editframe/cli` install the necessary packages. Open your terminal and run:

1. Create a new project using `@editframe/cli` and choose the "React" template:

```bash
npx @editframe/cli create
cd /path/to/your/project
```
2. Install the required packages  `framer-motion`:
```bash
npm install framer-motion
```

### Basic Setup

To start using Editframe React and Framer Motion, you'll need to import the required components and hooks in your React component:

- Update `src/Video.tsx`:

```jsx
import React from "react";

export const Video = () => {

    <>

        duration="2s"

        <Framer percentComplete={percentComplete} durationMs={durationMs} />
      </Timegroup>

  );
};
```

## Editframe React

Editframe React provides key components and hooks for managing animations and timing in your React applications.

```jsx
import React from 'react';
import { Timegroup, useTimingInfo } from '@editframe/react';
import { useAnimate } from 'framer-motion';
```

### Timegroup Component

The `Timegroup` component is a core element of Editframe React. It allows you to group and control the timing of child elements.

```jsx
<Timegroup
    className="bg-white w-[500px] h-[500px] overflow-hidden"
    mode="fixed"
    duration="2s"
    ref={ref}
>
  {/* Your animated content goes here */}
</Timegroup>
```

Key props:
- `id`: A unique identifier for the Timegroup
- `mode`: Determines how the animation behaves (e.g., "fixed" for a set duration)
- `duration`: The length of the animation
- `slot`: Specifies where the content should be rendered

### useTimingInfo Hook

The `useTimingInfo` hook provides timing information for your animations.

```jsx
const { percentComplete, durationMs, ref } = useTimingInfo();
```

- `percentComplete`: A value between 0 and 1 representing the animation progress
- `durationMs`: The total duration of the animation in milliseconds
- `ref`: A React ref to be attached to the Timegroup component

## Framer Motion

Framer Motion is a popular animation library for React that provides a simple API for creating complex animations.

### useAnimate Hook

The `useAnimate` hook is a powerful tool for creating and controlling animations in Framer Motion.

```jsx
const [scope, animate] = useAnimate();
```

- `scope`: A ref object that you can attach to the parent element of your animations
- `animate`: A function to define and control animations

### Animation Basics

Here's a simple example of how to create an animation with Framer Motion:

```jsx
useEffect(() => {
  animate(
    scope.current,
    { opacity: [0, 1] },
    { duration: 2 }
  );
}, [animate, scope]);
```

This animation fades in the element over 2 seconds.

## Integrating Editframe React with Framer Motion

To combine the power of Editframe React and Framer Motion, you can use the timing information from Editframe to control Framer Motion animations.

```jsx
useEffect(() => {
  const animation = animate(
    scope.current,
    { opacity: [0, 1] },
    { duration: durationMs / 1000 }
  );
  
  animation.pause();

  return () => animation.stop();
}, [animate, scope, durationMs]);

useEffect(() => {
  for (const animation of scope.animations) {
    animation.time = percentComplete * (durationMs / 1000);
  }
}, [scope.animations, percentComplete, durationMs]);
```

This code creates an animation, pauses it immediately, and then uses the `percentComplete` value from Editframe to precisely control the animation's progress.

## Troubleshooting

- If animations aren't working, check that all required packages are installed and imported correctly.
- Ensure that the `Timegroup` component has a valid `ref` prop connected to the `useTimingInfo` hook.
- Check the console for any error messages related to Framer Motion or Editframe React.

Combining Editframe React and Framer Motion allows for powerful, precisely-timed animations in your React applications. By leveraging the timing control of Editframe with the flexible animation capabilities of Framer Motion, you can create stunning visual experiences for your users.
