## Use CSS Variables

### Problem

Create animations that work regardless of timegroup duration. Hard-coding animation delays breaks when durations change.

### Solution

Use CSS variables: `--ef-duration`, `--ef-progress`, `--ef-transition-duration`, `--ef-transition-out-start`.

```css
animation-delay: calc(var(--ef-duration) - 2s);
width: calc(var(--ef-progress) * 100%);
animation: fade-out var(--ef-transition-duration) ease-in-out var(--ef-transition-out-start) paused;
```

**Note:** `--ef-progress` is stored as a number (0-1), not a CSS percentage. This allows it to work correctly in CSS calculations like `calc(var(--ef-progress) * 100%)` without being converted to pixel values based on element dimensions. When you need a percentage value, multiply by 100% in your CSS.
