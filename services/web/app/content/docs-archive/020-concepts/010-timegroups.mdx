# Timegroups

The fundamental organizing unit of an Editframe video is the timegroup. A timegroup can be used to declare several kinds of video timelines.

<EFTimegroup
  mode="sequence"
  duration="10s"
  className="w-[500px] h-[500px]"
>
  <EFTimegroup mode="fixed" duration="5s"></EFTimegroup>
  <EFTimegroup mode="fixed" duration="15s"></EFTimegroup>
</EFTimegroup>
