## Choose Between trimStart and sourceIn

**Problem:** You need to trim a video but aren't sure which approach to use.

**Solution:** Choose based on your mental model and use case:

**Use `trimstart`/`trimend` when:**
- Thinking "remove X seconds from edges"
- Building UI with sliders that adjust trim length
- Adjusting existing clip duration
- Working with durations rather than absolute times

**Use `sourcein`/`sourceout` when:**
- You know exact timestamps (e.g., from video editor)
- Referencing specific moments by timecode
- Need frame-perfect accuracy
- Working with timecode from video editing software

```jsx
// Relative: "Remove 2s from start, 3s from end"
<Video src="video.mp4" trimStart="2s" trimEnd="3s" />

// Absolute: "Show exactly seconds 5-15"
<Video src="video.mp4" sourceIn="5s" sourceOut="15s" />
```

Both produce the same visual result but differ in time tracking. See [Trimming Semantics](/docs/elements/video/explanation/trimming-semantics) for technical details.
